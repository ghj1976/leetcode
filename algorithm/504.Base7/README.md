# [504. Base 7](https://leetcode.com/problems/base-7/)

## 2019/06/07

### 题目 💗[easy]

Given an integer, return its base 7 string representation.

转成 7 进制

Example 1:

```bash
Input: 100
Output: "202"
```

Example 2:

```bash
Input: -7
Output: "-10"
```

Note: The input will be in range of [-1e7, 1e7].
